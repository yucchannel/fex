Welcome to FEX - Feature Extraction System

Your go-to system for real-time object recognition and classification using image processing techniques.

About FEX

FEX is a powerful feature extraction system that uses advanced image processing techniques like Sobel edge detection to detect and classify objects. It is ideal for applications like surveillance, automation, and object detection.

How to Use

To use FEX, simply install the system, add your dataset of labeled images, and run the program to start detecting objects in real-time through your webcam.

Installation

  • Clone the repository
  • Install the required dependencies using `pip install -r requirements.txt`
  • Run the system with `python main.py`

For more information, visit our GitHub repository.
